# ClimbQuest · CPT208 Coursework Portfolio

ClimbQuest is a coursework project website designed around the indoor climbing context, presenting a complete human-centered design workflow from problem definition to prototype evaluation.

## Project Overview

- **Module**: CPT208
- **Theme**: C2 – Go Climbers
- **Positioning**: An onsite support system for climbers and coaches
- **Website Type**: Static multi-page portfolio (HTML + CSS)

This project focuses on:

- Route discovery and guided challenges (Route Quest)
- Progress and achievement visualization (Progress Badges)
- Coaching feedback and learning support (Coach Feedback)

## Page Structure

- `index.html`: Home page and project summary
- `pages/motivation.html`: Project motivation
- `pages/research.html`: Academic and market research
- `pages/users.html`: User requirements and personas
- `pages/ideation.html`: Ideation and concept comparison
- `pages/prototype.html`: System design and prototype details
- `pages/evaluation.html`: Usability evaluation and reflection
- `pages/references.html`: References
- `pages/team.html`: Team members and contributions

## Run Locally

From the project root directory, run:

```bash
python -m http.server 5500
```

Then open in your browser:

```text
http://localhost:5500/
```

## Design and Visual Style

The current version uses a sporty, climbing-themed visual style:

- A GitHub icon button in the top navigation
- Climbing wall texture and climbing-hold background elements
- Route numbering and difficulty-style visual language in headings and cards
