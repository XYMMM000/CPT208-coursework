Prototype Overview

The ClimbQuest prototype is a mobile-first web application designed for use inside climbing gyms. It focuses on simplicity, fast navigation, and clear visual feedback so that users can access information quickly before, during, and after climbing attempts.

Feature 1: Route Quest

Users receive suggested route challenges based on their level and goals. The system helps transform route selection into a structured journey.

Feature 2: Progress Dashboard

A dashboard displays completed challenges, earned badges, and climbing streaks to encourage consistency and growth.

Feature 3: Feedback Cards

Coaches can provide short comments linked to attempts, helping users reflect on specific skills and next-step improvements.

System Architecture

The interface is structured into three core modules: route guidance, progress tracking, and coaching feedback. These modules are connected through a unified user profile and session-based interaction flow.
